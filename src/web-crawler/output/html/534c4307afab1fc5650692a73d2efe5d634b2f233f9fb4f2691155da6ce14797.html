COMMUNITY POLL
The conflict between the past and present needs a Revenant hunter to represent all Revenants.

From October 28 to November 11, help us build a one of a kind Revenant through a sample of the character creator options from CODE VEIN II. Choose from a base appearance, hairstyle, clothing, accessories, and cast your vote below to collectively create one Revenant who can save the world from destruction. The selections with the most total votes for each feature will be adopted to design our Revenant hunter which will show up in future CODE VEIN videos.  

This poll offers a brief look at the character creation feature in CODE VEIN II, with the full experience, including many more options, coming at launch on January 30, 2026.
