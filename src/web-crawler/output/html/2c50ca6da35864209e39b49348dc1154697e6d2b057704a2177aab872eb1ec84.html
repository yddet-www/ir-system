Gamer's companion & calendar With GamingBuddy you can: - Keep up with new releases - See OpenCritic scores & read reviews - Track your games (backlog, wishlist, ...) - Follow gaming events (Gamescom, awards etc) - Add countdown widgets to your Home Screen - Discover new games to play - Search the entire IGDB database - Share game details with friends (using universal links) - And much more! # GamingBuddy Unlimited GamingBuddy contains paid "Unlimited" version for more advanced features, data and widgets. The core experience is completely free and you can use the app without paying. # OpenCritic integration Find out how a new game fared with the critics and read reviews (Unlimited required). Also featuring Reviews tab to keep up with recent reviews from big websites # Available widgets: - Release countdown: Countdown to a single game release - Calendar: Games coming out this month (that you favorited) - Up Next: Showing your upcoming favorite games - Featured Up Next: Shows games from the "Featured" list (Unlimited required) - Featured Calendar: Shows games from the "Featured" list (Unlimited required) ---- Privacy Policy: https://impresskit.net/p/82640266-1a55-4ce6-a0ce-18b337db961f Terms of Use: https://www.apple.com/legal/internet-services/itunes/dev/stdeula/
